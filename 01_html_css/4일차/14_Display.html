강강조 작은영역 
        ------------------
        display : 화면에 보이는 성격변경(원래 가지고있는 성격을 바꾸는것x)
        - inline : 블럭성격, 거의(x)
        - block : 인라인을 블럭으로 변경할때 (가장 많이쓰는 경우)
                    화면에 보이기(js)
                    none : 화면에서 사라지기(js)
        - inline-block : 10%정도 씀
                        박스옆으로 나열
                        작은영역의 나열(상단유틸메뉴)

                        float : 15%
                        주(flex) : 80%
