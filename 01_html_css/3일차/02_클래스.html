        클래스 선택자
        별명/별칭
        한페이지에 여러번 사용 가능
        형식
        <태그 class="클래스명">
        style: .클래스명

        클래스명 규칙
        1. 영문자, 숫자, _, - 혼용가능
        2. 첫글자는 반드시 숫자제외
            주로 영문자 소문자
        3. 대소문자 구별한다
        4. 특수문자 한글 사용불가
        5. 두단어가 합쳐질경우
            1) 단어-단어 : ex) dog-cat
            1) 단어-단어 : ex) dog-cat
            1) 단어단어첫글자대문자 : ex) dogCat
        6.주로 영역(dive 집) 클래스를
        작성하는것이 좋다

        ex) 클래스: dog-cat
            아이디: dogCat
