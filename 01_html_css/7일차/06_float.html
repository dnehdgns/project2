        박스 옆으로 나열
        1. display:inline-block

        2. float : 15프로 정도
        띄워서 나열
        옆에 대상도 옆으로 따라서 나열된다.

        left는 띄워서 왼쪽부터 나열
        right는 띄워서 오른쪽부터 나열
        none는 적용되지않은것

        해제
        left : 왼쪽 해제
        right : 오른쪽 해제
        both : 둘다해제(얘를많이씀ㄴ)

        * 자식을 띄우면 부모가 자식을 인식하지 못한다 

        해결 

        1. 부모에 overflow:hidden 

        2. 부모에 ::after { content:""; display:block; clear:both } 

        주로 .clearfix::after { content:""; display:block; clear:both } 만들고 사용한다
