        transform:
        위치 : 자기 자신이 기준
        translate(x,y)
        translateX(x)
        translateY(y)

        크기 : 100% -> 1배
        자식들도 크기 적용된다
        scale(너비,높이)
        scaleX(너비)
        scaleY(높이)

        회전 : 
        rotate : 360도를 기준 deg->양수 음수사용
        양수         음수
            위(0)
        왼(-90) 오(90)
            아래(180)
        
        rotateX() x축기준 회전
        rotateY() y축기준 회전

        기울기 : 양수 음수 가능
        skew(x,y)
        skewX(xdeg)
        skewY(ydeg)
